TC39 与知识产权（IP）

# TC39 是如何运作的

TC39 是 Ecma 的一个技术委员会，它负责创建和维护 ECMA-262，即 JavaScript 编程语言的标准，以及一些其他与 JavaScript 相关的规范，例如 ECMA-402（『国际』）。

Kim Yang 2021/9/1 ECMA
成为 TC39 代表

# 成为 TC39 代表

TC39 是 Ecma International 的一部分。Ecma 成员资格是面向组织而非个人的。你需要作为成员组织加入 Ecma 或者作为一个 Ecma 成员组织的代表，才能参加线下的 TC39 会议。
